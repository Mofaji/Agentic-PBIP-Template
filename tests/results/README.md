# AI Test Results

This folder stores outputs generated by the markdown-based AI PBIP test runner.

## Files

- `test_results.md`: latest generated result (stable path)
- `latest_run.json`: metadata for the latest run
- `history/test_results_<UTC timestamp>.md`: immutable run history

## Result Source

Prompt file:
- `tests/prompts/PowerBI_Test.md`

Runner:
- `scripts/run-ai-md-test.ps1`

Workflow:
- `.github/workflows/ai-testing.yml`
